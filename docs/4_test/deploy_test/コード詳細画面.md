# 対象

/codes/[code_id]/detail

## テスト

### コード詳細機能

- 表示
- [ ] 公開設定
  - [ ] コードアイコンが表示される
  - [ ] コードタイトルが表示される
  - [ ] 公開日と更新日が表示される
  - [ ] お気に入り数が表示される
  - [ ] コメント数が表示される
- [ ] 非公開設定
  - [ ] 「コードが見つかりませんでした」と表示される


### コメント一覧機能

- 表示
- [ ] コメントが無い場合
  - [ ] まだコメントはありませんと表示される
- [ ] コメントがある場合
  - [ ] ヘッダーに「ユーザー名」「アイコン」「コメント日」が表示される
  - [ ] 自分のコメントの場合「設定」ボタンが表示される

- 削除
- [ ] 自分のコメントの場合
  - [ ] 削除される
- [ ] 0件の場合
  - [ ] 「まだコメントはありません」と表示される

- 遷移
- [ ] アイコンから「ユーザー詳細画面」に遷移できる

### コメントモーダル機能

- 起動
  - [ ] サイドバー＞「コメントする」から起動する
- コメント作成
- [ ] 0件から作成
  - [ ] 「まだコメントはありません」からコメントが表示される
- [ ] コメントを5000文字で投稿する
  - [ ] 成功する
  - [ ] 投稿日・ユーザー名・アイコンが表示される
  - [ ] ダッシュボード＞コメントに追加されている
- [ ] コメントを0文字で作成する
  - [ ] 「コメントの投稿に失敗しました」というトーストが表示される。
- [ ] コメントを5001文字で投稿する
  - [ ] 入力できない


### ファイル一覧機能

- 表示
- [ ] 何もファイルが無い場合
  - [ ] 「ファイルがありません」と表示される
- [ ] ファイルがある場合
  - [ ] ファイル一覧が表示される



### その他機能

- [ ] 編集ボタンから編集画面に遷移できること
